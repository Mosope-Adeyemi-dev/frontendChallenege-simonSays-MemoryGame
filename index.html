Simon Says

Press A Key to Start

  1. This is a clone of the popular simon say's game
  2. Its major aim is to help build children ability to commit things to memory
  3. When you start the game, it will start of by showing you a color e.g blue
  4. Once you press blue, it will show you another color, lets say green
  5. You know have to remember the sequence from when the game started
  6. I.e you would have to press Blue first followed by the newly shown color which was blue, and so on
  7. The goal is to remember the pattern
  8. Have fun playing!!
